# 🌸 ZOE 內在阻礙地圖測驗

給「ZOE 療癒與內在轉化研究院(心靈 Love Beauty 香氣感官所)」客人的免費自我覺察測驗。
3 分鐘、14 題,產出個人化的「內在阻礙地圖」:

- 🌊 **情緒負載指數** — 由情緒種類、強度、持續時間、影響層面計算
- 🔍 **五向度覺察力** — 行為/情緒/思維/身體/能量,雷達圖呈現,指出最需要關注的向度
- 🔥 **行動準備度** — 由決心程度與因應風格計算
- 🦋 **四種內在類型** — 破繭者/休眠的火山/顯化行動者/靜水深流

## 特色

- 純前端單頁(vanilla HTML/CSS/JS),零依賴、零後端,開 GitHub Pages 就能用
- 作答資料只留在使用者裝置,不上傳
- 結果頁一鍵「複製測驗結果」,客人可直接貼進 Line 對話
- 手機優先設計

## 自訂

所有對外連結集中在 `index.html` 的 `CONFIG` 區塊:

```js
const CONFIG = {
  LINE_URL: "",       // 官方 Line 邀請連結(填入後結果頁會出現 Line 按鈕)
  BOOKING_URL: "...", // 預約表單連結
};
```

題目與計分邏輯都在 `index.html` 內的 `QUESTIONS`、`DIM_INFO`、`TYPES` 與 `calc()`,直接改即可。
